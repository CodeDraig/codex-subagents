# Journal Submission Workflows

## Overview

Prepare scholarly submission packages by matching journal instructions, manuscript components, metadata, ethics statements, and revision responses.

## Workflow

1. Restate journal, article type, submission stage, author roles, deadline, and instructions version.
2. Extract manuscript, metadata, reporting guideline, ethics, disclosure, data, and file requirements.
3. Build a submission checklist with owner, status, evidence, and unresolved questions.
4. For revisions, map reviewer comments to responses, manuscript changes, and evidence.
5. Produce package gaps and next actions.

Use `references/journal-submission-checklist.md` for submission readiness.

## Output Contract

Return exactly: `Submission Scope`, `Journal Requirements`, `Package Checklist`, `Metadata Gaps`, `Ethics And Disclosure Items`, `Reviewer Response Matrix`, `Next Actions`.

## Stop Conditions

Stop when asked to fabricate data, ghostwrite undisclosed authorship, hide conflicts, bypass ethics requirements, submit without authorization, or misrepresent reviewer responses.
